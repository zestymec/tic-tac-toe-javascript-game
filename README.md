# Tic-Tac-Toe Game (JavaScript)

Classic Tic-Tac-Toe game made from scratch with HTML, CSS & JavaScript.

## Features
- Player X & O turns
- Real-time win detection (rows, columns, diagonals)
- Disable clicked boxes
- Winner announcement with restart option
- Responsive design

## How to Play
1. Clone repo
2. Open index.html in browser
3. Click boxes to play!
